## How to Run Locally

1. Install `rvm` (https://rvm.io)
2. Install ruby version 2.2.5 `rvm install 2.2.5`
3. Install bundler `gem install bundler`
4. Run `bundle install`
5. Copy `secrets.sample.yml` to `secrets.yml`
6. Run `rails s`
